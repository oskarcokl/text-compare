# Text Compare

A lightweight, client-side text diff tool. Paste two texts and instantly see what changed, with line-level and word-level highlighting.

**Live:** https://oskarcokl.github.io/text-compare/

## Features

- Unified and side-by-side diff views
- Word-level inline highlighting
- Pure HTML/CSS/JS — no build step, no dependencies

## Run locally

```bash
python3 -m http.server 8080
```

Then open http://localhost:8080.
